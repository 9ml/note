# 扩展运算符的应用

- 数组的合并

```javascript
const info = ['9ml', 18, '男']
const like = ['万能青年旅店', '草东没有派对', '寂寞夏日']
// ES5 concat
const userA = info.concat(like) // ['9ml', 18, '男', '万能青年旅店', '草东没有派对', '寂寞夏日']
// ES6 扩展运算符
const userB = [...info, ...like] // ['9ml', 18, '男', '万能青年旅店', '草东没有派对', '寂寞夏日']
```

- 数组的克隆

```javascript
const user = ['9ml', 18, '男']
// 克隆
const newUser = [...user]
```

- 将伪数组转为真正的数组

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body>
  <div></div>
  <div></div>
  <div></div>
  <script>
    const dvs = document.querySelectorAll('div')
    console.log(dvs) // NodeList 原型是对象
    const dvArr = [...dvs]
    console.log(dvs) // 真正的数组，可以使用数组的API
  </script>
</body>
```
