## 关键字和保留字

### 关键字的定义和特点

- 关键字`keyword`：被`Java`语言赋予了特殊含义，用做专门用途的字符串（单词）
- 特点：关键字中所有的字母都为小写
- [官方网址](https://docs.oracle.com/javase/tutorial/java/nutsandbolts/_keywords.html)
- 用于定义数据类型的关键字：
  - `class`
  - `interface`
  - `enum`
  - `byte`
  - `short`
  - `int`
  - `long`
  - `float`
  - `double`
  - `char`
  - `boolean`
  - `void`
- 用于定义流程控制的关键字
  - `if`
  - `else`
  - `switch`
  - `case`
  - `default`
  - `while`
  - `do`
  - `for`
  - `break`
  - `continue`
  - `return`
- 用于定义访问权限修饰符的关键字
  - `private`
  - `protected`
  - `public`
- 用于定义类、函数、变量修饰符的关键字
  - `abstract`
  - `final`
  - `static`
  - `synchronized`
- 用于定义类与类之间关系的关键字
  - `extends`
  - `implements`
- 用于定义建立实例及引用实例，判断实例的关键字
  - `new`
  - `this`

- 用于异常处理的关键字
  - `try`

- 其他修饰符关键字
  - `native`
  - `stricfp`

  - `assert`
- *用于定义数据类型值的字面值*
  - `true`
  - `false`
  - `null`

### 保留字

- `Java`保留字：现有`Java`版本尚未使用，但以后版本可能会作为关键字使用，自己命名标识符时要避免使用这些保留字：
  - `goto`
  - `const`

## 标识符

- 标识符`Identifier`
  - `Java`对各种变量、方法和类等要素命名时使用的字符序列称为标识符
  - 技巧：凡是自己可以起名字的地方都叫标识符
- 定义合法标识符规则：如果不遵守以下规则无法编译
  - 由`26`个英文字母大小写，`0-9`，`_`或`$`组成
  - 数字不可以开头
  - 不能使用关键字和保留字，但能包含关键字和保留字
  - `Java`严格区分大小写，长度无限制
  - 标识符不能包含空格

### 命名规范

- `Java`中的名称命名规范：
  - 包名：多单词组成时所有字母都小写：`xxxyyyzzz`
  - 类、接口名：多单词组成时所有单词的首字母大写：`XxxYyyZzz`大驼峰命名
  - 变量、方法名：多单词组成时，第一个单词首字母小写，后面的单词首字母大写：`xxxYyyZzz`小驼峰命名
  - 常量名：所有字母都大写，多个单词时每个单词用下划线连接：`XXX_YYY_ZZZ`
- 注意：
  - 为了提高可阅读性，起名时尽量有意义
  - `Java`采用`unicode`字符集，因此标识符也可以使用汉字命名，但是不建议使用
  - 详见《代码整洁之道》
- 示例

```java
class IdentifierTest {
  public static void main(String[] args) {
    int myNumber = 1024;
    System.out.println(myNumber);
  }
}
```

## 变量

> - 基本数据类型
> - 基本数据类型变量间转换
> - 基本数据类型与`String`间转换
> - 进制与进制间转换

- 变量的概念：
  - 内存中的一个存储区域
  - 该区域的数据可以在同一类型范围内不断变化
  - 变量是程序中最基本的存储单元，包含**变量类型、变量名和存储的值**
- 变量的作用：
  - 用于在内存中保存数据
- 使用变量注意：
  - `Java`中每个变量必须先声明赋值，后使用
  - 使用变量名来访问这块区域的数据
  - 变量的作用域：其定义所在的一对`{}`内
  - 变量只有在其作用域内才有效
  - 同一作用域内，不能定义重名的变量
- 声明变量
  - 语法：`<数据类型>` `<变量名称>;`
  - 如：`int num;`
- 变量的赋值：
  - 语法：`<变量名>` `=` `<值>;`
  - 如：`num = 100;`
- 声明并赋值变量
- 语法：`<数据类型>` `<变量名称>` `=` `<值>;`
- 如：`int num = 100;`
- 示例

```java
class VariableTest {
  public static void main(String[] args) {
    // 变量的定义
    int userAge = 12;
    // 变量的使用
    System.out.println(userAge)

    // 使用 myAge 之前并未定义 myAge，编译报错
    System.out.println(myAge)
    // 变量的声明
    int myAge;
    // 使用 myAge 之前并未赋值 myAge，编译报错
    System.out.println(myAge)
    // 变量的赋值
    myAge = 18;
    // 作用域内未找到，编译报错
    System.out.println(myClass)
    // 同一作用域内变量重名，编译报错
    int myAge = 20
  }
}

class void method() {
  int myClass = 1
}
```

### 变量的分类

- `Java`对于每一种数据都定义了明确的具体数据类型（强类型语言），在内存中分配了不同大小的内存空间

#### 按数据类型

- 按数据类型分为：
  - 基本数据类型`primitive type`：
    - 整型：`byte`、`short`、`int`、`long`
    - 浮点型：`float`、`double`
    - 字符型：`char`
    - 布尔型：`boolean`
  - 引用数据类型`reference type`：
    - 类`class`
    - 接口`interface`
    - 数组`array`

#### 按声明位置

- 按声明的位置的不同分为：
  - 成员变量：在方法体外，类体内声明的变量
  - 局部变量：在方法体内声明的变量
- 注意：二者在初始化值方面的异同：
  - 同：都有生命周期
  - 异：局部变量除形参外，需显式初始化

#### 整型

- 整数类型：`byte`、`short`、`int`、`long`
- `Java`各整数类型有固定的表数范围和字段长度，不受具体`OS`的影响，以保证`Java`程序的可移植性
- `Java`的整型常量默认为`int`，声明`long`型常量须后加`l`或`L`
- `Java`程序中变量通常声明为`int`型，除非不足以表示较大的数才使用`long`
- 区别：

|  类型   |    占用存储空间    |            表数范围            |
| :-----: | :----------------: | :----------------------------: |
| `byte`  | `1`字节 = `8bit`位 |         `-128` ~ `127`         |
| `short` |      `2`字节       |      `-2**15` ~ `2**15-1`      |
|  `int`  |      `4`字节       | `-2**31` ~ `2**31-1`(约`21`亿) |
| `long`  |      `8`字节       |      `-2**63` ~ `2**63-1`      |

- `bit`计算机中最小的存储单位
- `byte`计算机中基本存储单元
- 示例：

```java
class VariableTest {
  public static void main(String[] args) {
    // 整型：byte(-128 ~ 127) / short / int / long
    byte b1 = 12;
    byte b2 = -128;
    // b2 = 128 // 超出范围，编译不通过

    // 声明 long 型变量，必须以 l 或者 L 结尾
    long l1 = 451245456412L;
    long l2 = 689456568454l;
  }
}
```
